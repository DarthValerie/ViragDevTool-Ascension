# ViragDevTool (Ascension)

A compatibility port of **ViragDevTool 0.22** from **World of Warcraft Retail (8.0)** to **Wrath of the Lich King 3.3.5**, compatible with **Ascension WoW**.

ViragDevTool is a debugging and inspection tool for addon developers, allowing you to inspect tables, call functions, monitor events, and log function calls in-game.

## Features

- ✅ Search
- ✅ Function calls
- ✅ Function logger
- ✅ Event monitor
- ✅ History
- ✅ Expand/Collapse tables

## Compatibility Fixes

This port includes fixes for Wrath/Ascension compatibility, including:

- HybridScrollFrame compatibility
- Main table rendering fixes
- Sidebar rendering fixes
- Column sizing fixes
- Resize handling
- Function logger compatibility
- Event monitor compatibility
- Various API and UI compatibility fixes

## Installation

1. Download the latest release.
2. Extract `ViragDevTool` into your `Interface/AddOns` folder.
3. Restart the game if installing or `/reload` if updating addon.
