# ViragDevTool (Ascension)

A port of ViragDevTool 0.22 from World of Warcraft Retail (8.0)
to Wrath of the Lich King 3.3.5 for Ascension.

## Working

- Main window
- Sidebar
- Search
- Function calls
- Function logger
- Event monitor
- History
- Expand/Collapse
- Scrollbars
- Resizing
